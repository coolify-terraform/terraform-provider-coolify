# Support

## Getting Help

- **Documentation**: [Terraform Registry docs](https://registry.terraform.io/providers/coolify-terraform/coolify/latest/docs)
- **Issues**: Search [existing issues](https://github.com/coolify-terraform/terraform-provider-coolify/issues) or open a new one
- **Coolify**: For Coolify application questions, see the [Coolify documentation](https://coolify.io/docs)

## Reporting Bugs

When reporting a bug, please include:

1. Terraform and provider versions (`terraform version`)
2. Relevant Terraform configuration (sanitized of secrets)
3. Expected vs. actual behavior
4. Steps to reproduce

## Security Issues

For security vulnerabilities, see [SECURITY.md](SECURITY.md).
