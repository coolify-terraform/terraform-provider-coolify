# Coolify Provider

The Coolify provider allows you to manage resources in a [Coolify](https://coolify.io/) instance, the open-source, self-hosted Platform-as-a-Service.

## Authentication

The provider requires a Coolify API token. Generate one in the Coolify UI under **Keys & Tokens > API tokens**.

You can configure credentials via provider attributes or environment variables:

- `COOLIFY_ENDPOINT` for the API base URL
- `COOLIFY_TOKEN` for the API token

## Example Usage

```terraform
provider "coolify" {
  endpoint = "http://coolify.example.com:8000"
  token    = "your-api-token" # Or set COOLIFY_TOKEN env var
}
```
